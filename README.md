## 🎨 Design System

### Colors
- Primary Steel: `#5B74A6`
- Deep Navy: `#1E3340`
- Ice White: `#F8F9FA`

### Typography
- Body: Inter (Google Fonts)
- Buttons: Poster Gothic Cond ATF Medium

### Signature Design
- 45° corner cut-off on cards & buttons
- Auto-detect dark mode
- Microanimations on hover
